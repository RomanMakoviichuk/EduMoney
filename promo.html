Скачать WP тему »

Инструкция

  1. Скачиваете архив с WordPress-темой.
  2. На вашем WP-сайте заходим в раздел: "Темы" - Добавить новую - Загрузить новую.
  3. Заливаем скачанный вами файл "extraessay_wp_theme". Нажимаем "Установить".
  4. Активируем данную тему на вашем сайте.
  5. Переходим в раздел "Внешний Вид" - Редактор.
  6. Переходим в заголовок (header.php).
  7. Находим в коде следующее:
    <script type='text/javascript'>
    /*
    * WGWP_KEY Your Key
    */
    var WGWP_KEY = 'Your Key';
    </script>
     В строку var WGWP_KEY = 'Your Key'; вставляем ваш ключ веб-мастера между ковычек ' '.
    Ваш ключ будет приведён ниже:
    var WGWP_KEY = 'b8a17bc8ba1fd9bf0d8f7f5cae7265f0';
  8. Вуаля! WP-тема работает исправно.
